# Example 1

Yes - You have a desase, No - You don't have a desease P - You have positive test result, N - You have negative test result

Probability of get the desease is 1% ( P(Yes) = 1% ).

The test is correct in 95% of examples when it is positive ( This mean probability that you get positive result in the test when you have desease is: P(P|Yes) = 95% ), and 97% accurate when it's negative (This mean probability that you get negative result in the test when you don't have desease P(N|No) = 97% )

  • What is the probability if you have a desease when you get positive result in the test ( P(Yes∣P)P(Yes|P)P(Yes∣P) ) ?.

P(Yes)= 1%
P(No)= 1 - P(Yes) = 99%
P(P|Yes)= 95%
P(N|No)= 97%
P(P|No)= 3% - Why? Because probability of geting positive result when you don't have desease is a false positive result. When test is accurate in 97% in negative, meanse that in 3% it shows positive.

P(Yes∣P)=P(P∣Yes)∗P(Yes)P(P)P(Yes|P)=\dfrac{P(P|Yes)*P(Yes)}{P(P)}P(Yes∣P)=​P(P)​​P(P∣Yes)∗P(Yes)​​

P(P)=P(P∣Yes)∗P(Yes)+P(P∣No)∗P(No)=95%∗1%+3%∗99%=0.0392P(P) = P(P|Yes)*P(Yes)+P(P|No)*P(No)=95\%*1\%+3\%*99\%=0.0392P(P)=P(P∣Yes)∗P(Yes)+P(P∣No)∗P(No)=95%∗1%+3%∗99%=0.0392

P(Yes|P)=\dfrac{95\%*1\%}{0.0392}=0.2423=24.23%
