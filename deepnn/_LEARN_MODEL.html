Adding more layers usually doesn’t improve model. If we create a 56-layer model it would be worse than 20-layer model.

What would happende if we create something that skip

This is used in the ResNet network, that each layer is the ResNet Block.

class ResBlock(nn.Module):
    def __init__(self, nf):
        super().__init__()
        self.conv1 = conv_layer(nf,nf)
        self.conv2 = conv_layer(nf,nf)
        
    def forward(self, x): return x + self.conv2(self.conv1(x))

def conv2(ni,nf): return conv_layer(ni,nf,stride=2)
