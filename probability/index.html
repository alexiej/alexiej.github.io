NominalOrdinalIntervalRatioscale
Description:When the value doesn't have any connection with the data. It is just for codification.When values have some order.Numeric values with 0, but scale is not equal (4 is not equal 2 * 2)There is nominal difference between values (4 is equal 2 * 2)
Example:- gender (1-male, 0-female)- degree (A,B,C,D,E)
- position (1st,2nd,3rd)		- Faranheit scale- Celcius scale
